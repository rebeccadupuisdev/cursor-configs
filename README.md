# .cursor Repository — Personas, Rules & Commands

This repository contains **Cursor personas, rules, and commands** to configure AI behavior.

## Folder Overview

- `personas/` — Defines AI behavior for tasks  
- `rules/` — Optional constraints or style guides applied alongside personas  
- `commands/` — Quick aliases or triggers to activate personas/rules  

For detailed instructions, see the READMEs in each folder:

- [Personas Guide](personas/README.md)  
- [Rules Guide](rules/README.md)  
- [Commands Guide](commands/README.md)
